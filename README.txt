===========================================
  FIFA World Cup 2026 - Telegram Reminder Bot
===========================================

HOW TO RUN (Windows):
----------------------
1. Make sure Python is installed (python.org)

2. Open Command Prompt (CMD) in this folder
   - Hold Shift + Right-click in the folder
   - Select "Open PowerShell/Command Prompt here"

3. Install dependencies (only once):
   pip install -r requirements.txt

4. Run the bot:
   python bot.py

5. You will see:
   FIFA Reminder Bot started! XX reminders scheduled.

6. Keep the CMD window open. You will get a
   Telegram message 30 minutes before every match.

7. To stop: press Ctrl + C

HOW TO RUN (Mac/Linux):
------------------------
1. Open Terminal in this folder
2. pip install -r requirements.txt
3. python3 bot.py

NOTE:
- Your PC must be ON for reminders to work
- All times are shown in Bangladesh Time (BD)
- Bot token is in config.py — keep it private!
